# Deep-Learning-Experiments-

This repository contains a set of deep learning experiments implemented using **Keras / PyTorch**.  
The aim is to understand core deep learning concepts through practical implementation.

---

## 📌 Experiments

| No. | Experiment |
|-----|-----------|
| 1 | Loading dataset into Keras/PyTorch and creating train-test splits |
| 2 | Creating functions to compute various loss functions |
| 3 | Feeding data to pretrained neural networks and making predictions |
| 4 | Implementing regression using a deep neural network |
| 5 | IMDB movie review classification (Binary Classification) |
| 6 | Reuters dataset classification (Multiclass Classification) |
| 7 | MNIST digit classification using CNN |
| 8 | Transfer Learning using pretrained models |
| 9 | Temperature forecasting using RNN |
| 10 | GAN implementation on a suitable dataset |

---

## 🛠️ Tech Stack

- Python  
- TensorFlow / Keras  
- PyTorch  
- NumPy  
- Pandas  
- Matplotlib  
- Scikit-learn

## 🎯 Objective
Understand deep learning fundamentals
Implement neural network architectures
